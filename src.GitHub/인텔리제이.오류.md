## Invalidate Caches
- 인텔리제이가 "이 폴더 안에 뭐가 있는지" 다시 스캔하게 만드는 방법입니다.
- 왼쪽 상단 [File] -> [Invalidate Caches...] 를 누릅니다.
- 뜨는 팝업에서 모든 체크박스를 선택하고 **[Invalidate and Restart]**를 누르세요.
- 인텔리제이가 꺼졌다가 다시 켜지면서 하단에 **"Scanning files to index"**라는 로딩 바가 생길 겁니다. 이 로딩이 끝날 때까지 잠시만 기다려 보세요.
